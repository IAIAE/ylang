# this
Ylang是一个前端javascript打包器，也可以认为是“微前端”的一种落地实现。它基于webpack，在此基础上提供了沙箱环境，进而落地“微前端”。注意它只针对浏览器环境下的js工程使用，不适用于nodejs工程。
